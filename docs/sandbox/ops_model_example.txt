Access_Info --- The class Access_Info describes an operational access endpoint or representation associated with a PDS4 product or file.
  - access_uri: ASCII_AnyURI [0..1]
  - access_method: ASCII_Short_String_Collapsed [0..1]
  	Permissible values:
  	-- https --- The access method uses HTTPS.
  	-- s3 --- The access method uses an Amazon S3 URI or compatible object-storage URI.
  	-- native --- The access method uses a native or system-local path.
  	-- internal --- The access method is intended for internal operational use.
  
  - product_representation_type: ASCII_Short_String_Collapsed [0..1]
    Permissible values:
    - browse --- The access endpoint provides a browse or preview representation.
    - download --- The access endpoint provides a downloadable representation.
    - compressed --- The access endpoint provides a compressed representation.
    - cloud-native --- The access endpoint provides a cloud-native representation.
    - alternate --- The access endpoint provides an alternate representation.
		- primary --- The access endpoint provides the primary representation.

  - is_user_visible: ASCII_Boolean [0..1]
  - authorization_required: ASCII_Boolean [0..1]
  - access_note: ASCII_Text_Preserved [0..1]
